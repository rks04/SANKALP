# Sankalp Frontend

A premium, React-based user interface built with Vite, Tailwind CSS v4, and Framer Motion for buttery-smooth animations.

## Features
- **Glassmorphic Aesthetic**: Deep, warm themes with soft glows.
- **Theme Toggling**: Seamless light and dark modes with a custom animated toggle.
- **Responsive Pagination**: Efficiently browse through hundreds of tasks.

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev -- --host
   ```
*(Running with `--host` exposes the server for remote usage on your local network).*
