# SANKALP - AI Project Manager

## Main Motive
**Sankalp** was created to elegantly structure the chaos of team meetings. Often, meetings generate brilliant ideas and action items that are quickly lost in unstructured notes. Our mission is to seamlessly bridge the gap between human conversation and actionable engineering tasks, ensuring that every idea is captured, prioritized, and assigned with zero manual overhead. 

## Key Features
- **AI-Powered Task Generation**: Uses Google's advanced Gemini AI to instantly parse messy meeting notes and extract structured, prioritized tasks.
- **Premium Glassmorphic UI**: A stunning, buttery-smooth interface built with React, Vite, and Tailwind CSS v4 that feels incredibly satisfying to use.
- **Dynamic Dark/Light Themes**: A custom built, animated theme toggle allowing you to switch between a clean light mode and a calm, warm charcoal dark mode.
- **Robust Backend**: A high-performance FastAPI server featuring rate-limiting and paginated database queries for secure, scalable operation.
- **Remote Ready**: Fully configured for local network collaboration out of the box, allowing teams to generate tasks from multiple devices simultaneously.
- **CSV Export & Filtering**: Instantly filter tasks by owner, priority, and status, and export your sprints directly to CSV.

## Project Structure
- `/frontend`: React application built with Vite and Tailwind CSS v4. Features a premium, glassmorphic UI.
- `/backend`: FastAPI application powered by Python and Google's Gemini AI. 

## Tech Stack
**Frontend:**
- React (UI Framework)
